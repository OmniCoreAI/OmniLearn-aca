# Maintenance Overview

Keeping your self-hosted OmniLearn instance healthy requires periodic maintenance. The OmniLearn CLI provides built-in tools for the most common maintenance tasks.

## Maintenance Tasks

### [Updates](/self-hosting/maintenance/updates)

Keep OmniLearn up to date with the latest features, improvements, and security patches using `omnilearn update`.

### [Backups](/self-hosting/maintenance/backups)

Protect your data with regular backups of the database and uploaded content using `omnilearn backup`.

### [Monitoring](/self-hosting/maintenance/monitoring)

Monitor the health and performance of your instance using `omnilearn status` and `omnilearn logs`.

### [Troubleshooting](/self-hosting/maintenance/troubleshooting)

Diagnose and resolve common issues with your self-hosted instance.
