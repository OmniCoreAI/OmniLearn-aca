# Developers

Welcome to the OmniLearn developer documentation. This section covers everything you need to understand, build, and contribute to the OmniLearn platform.

## Project Structure

OmniLearn is organized as a monorepo containing four main applications:

| App | Technology | Description |
| --- | --- | --- |
| **web** | Next.js (React) | Frontend web application |
| **api** | FastAPI (Python) | Backend REST API |
| **collab** | Hocuspocus | Real-time collaboration WebSocket server |
| **cli** | Commander (Node.js) | Command-line interface for development and management |

All four apps live under the `apps/` directory in the [OmniLearn repository](https://github.com/omnilearn/omnilearn).
